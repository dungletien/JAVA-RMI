# Xây dựng chương trình quản lí sinh viên sử dụng kĩ thuật phân tán RMI

## Deployment
B1: Kết nối database <br>
B2: Biên dịch toàn bộ chương trình <br>
    ``javac *.java
<br>B3: Bắt đầu với registry<br>
    ``start registry
<br>B4: Chạy serve <br>
    ``javac RMIServe.java
<br>B5: Chạy Client<br>
    ``javac RMIClient.java
